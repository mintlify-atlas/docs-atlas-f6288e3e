## Overview

Boundary nodes serve as the **HTTP gateway** between external clients and the Internet Computer network. They provide API access, load balancing, caching, and security for incoming requests while routing them to appropriate subnet replicas.

## Architecture

```
Client Requests (HTTP/HTTPS)
         |
         v
  +--------------+
  | Boundary Node|
  |  (ic_boundary)|
  +--------------+
         |
         +--- TLS Verification
         +--- Request Validation  
         +--- Caching Layer
         +--- Rate Limiting
         +--- Subnet Routing
         |
         v
  Replica Nodes (QUIC/TLS)
```

### Core Responsibilities

1. **Protocol Translation**: HTTP/HTTPS to replica-native protocols
2. **Request Routing**: Canister-to-subnet mapping
3. **Load Balancing**: Health-aware replica selection
4. **Caching**: Response caching for query calls
5. **Security**: Rate limiting, DDoS protection, TLS termination
6. **Observability**: Metrics, logging, request tracing

### 4. Registry Integration

Maintains subnet and node information:

```rust
pub struct RegistrySnapshot {
    pub nns_public_key: Vec<u8>,
    pub subnets: Vec<Subnet>,
    pub routing_table: BTreeMap<CanisterId, SubnetId>,
}

pub struct Subnet {
    pub id: SubnetId,
    pub subnet_type: SubnetType,
    pub nodes: Vec<Node>,
}
```

**Update Cycle**:
1. Registry client polls local store (every 5 seconds)
2. Snapshotter creates new snapshot from registry
3. Health checker validates node availability
4. Persister updates routing table
5. Atomic swap of active snapshot

### 5. Health Checking

Continuous monitoring of replica health:

```rust
pub struct Checker {
    http_client: Arc<dyn Client>,
    timeout: Duration,
}
```

**Health Algorithm**:
```rust
impl Health for ProxyRouter {
    fn health(&self) -> ReplicaHealthStatus {
        let healthy_subnets = subnets.filter(|subnet| {
            let healthy_ratio = healthy_nodes / total_nodes;
            healthy_ratio >= self.nodes_per_subnet_alive_threshold
        }).count();
        
        let subnet_ratio = healthy_subnets / total_subnets;
        if subnet_ratio >= self.subnets_alive_threshold {
            ReplicaHealthStatus::Healthy
        } else {
            ReplicaHealthStatus::CertifiedStateBehind
        }
    }
}
```

Default thresholds:
- Subnets alive: **51%**
- Nodes per subnet alive: **66.66%**
